Online Sync

ข้อมูลกลาง sync ผ่าน cloud; เปิดจากคอม มือถือ หรือเครื่องอื่นได้ด้วย URL เดียวกัน

อ่านและบันทึกข้อมูลได้ทุกเครื่องผ่าน cloud sync. GitHub token ด้านบนเป็นแค่แบ็กอัพสำรองกรณีอยาก mirror กลับ repo. ปุ่มคัดลอกลิงก์ตั้งค่าเครื่องอื่นจะใส่ token ใน URL hash เพื่อความสะดวก ห้ามส่งลิงก์นั้นให้คนอื่น
TradingView Integration

เปิด marker-only integration แล้ว: Beer Top 100 ยังเป็นสมุดจดหลัก ส่วน TradingView ใช้แสดงหมุดเล็กบนแท่งเทียนตามวันที่บันทึก

บน TradingView ให้กดปุ่ม Beer Notes แล้วเลือก Copy Pine markers หรือ Try place on chart. ข้อมูลใน marker จะรวมทั้งมุมมองของคุณ ข่าวที่ Beer ใช้ และบทวิเคราะห์/ความเห็นของ Beer โดยไม่กางเป็นกล่องใหญ่บนกราฟ
