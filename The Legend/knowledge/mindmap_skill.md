# Skill: Lesson Mind Map Distillation

นี่คือวิธีที่ต้องใช้กับ The Legend ทุกบทก่อนทำบทเรียนรูปแบบอื่น

## เป้าหมาย
เปลี่ยน transcript ยาว ๆ ให้เป็น mind map ที่เห็นว่าอาจารย์พูดเรื่องอะไรบ้าง แต่ละเรื่องโยงกันอย่างไร และสุดท้ายบทนั้นต้องพาผู้เรียนไปจบที่ข้อสรุปอะไร

## ขั้นตอน
1. อ่าน transcript ทั้งบท ไม่ดึงแค่หัวข้อจากชื่อวิดีโอ
2. หาแกนกลางของบทว่า “บทนี้ต้องการเปลี่ยนความคิดอะไร”
3. แตกกิ่งระดับ 1 เป็นหัวข้อใหญ่ที่อาจารย์พูดจริง
4. แตกกิ่งระดับ 2 เป็นรายละเอียด เหตุผล ตัวอย่าง ประสบการณ์ หรือ framework
5. แตกกิ่งระดับ 3 เป็น action/checklist ที่ผู้เรียนต้องทำ
6. สรุปปลายทางว่า ถ้าเข้าใจบทนี้จริง ผู้เรียนต้องคิด/ทำ/ตัดสินใจต่างไปอย่างไร
7. ทำเป็นทั้ง visual mind map และ outline อ่านง่าย

## รูปแบบผลลัพธ์ของแต่ละบท
- Central Idea
- Big Branches
- Deep Branches
- Key Stories / Examples
- Decision Rules
- Final Destination
- Questions for Patiphan

## กฎสำคัญ
- ห้ามสรุปแบบเรียงความอย่างเดียว
- ห้ามตัดรายละเอียดที่เป็นเส้นทางความคิดออก
- ต้องบอกให้ได้ว่า “ทำไมกิ่งนี้ถึงโยงกับกิ่งนั้น”
- ต้องมีปลายทางของบทเสมอ
- ถ้าข้อมูล transcript เพี้ยน ให้ตีความอย่างระวังและระบุว่าเป็นการอ่านจากบริบท
